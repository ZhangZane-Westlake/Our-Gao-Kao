# Our-Gao-Kao

《一模前30天》是一款像素风高三生活模拟与轻度策略选择网页游戏。玩家从一模前 30 天开始，每天在学习、休息、人际、家庭沟通和短暂逃避之间做选择。游戏不追求“刷高分爽文”，而是让玩家看见：同样是努力，不同节奏会把人带向不同状态。

## 项目定位

- 类型：像素风高三生活模拟 + 轻度策略选择
- 周期：30 天试玩版
- 平台：浏览器网页
- 技术栈：Vite + React + TypeScript
- 核心体验：每个选择都有短期收益和长期代价

## 当前玩法

首版已实现完整可玩循环：

- 选择 5 种初始角色之一：稳定型、偏科型、压力型、松散型、逆袭型
- 每天分为 3 个时段：白天、晚自习、睡前
- 每个时段选择一个行动
- 行动影响 6 个可见属性：成绩、精力、心态、健康、人际、家庭
- 隐藏变量包括：专注力、压力、手机依赖、抗挫力
- 学习收益会受精力、心态、健康、压力和专注力影响
- 第 10、20、30 天触发考试
- 随机事件围绕朋友、父母、老师、失眠、排名、手机和考前情绪展开
- 第 30 天考试结束后进入结局
- 本地存档使用 localStorage 自动保存

## 内容结构

核心内容集中在 `src/content.ts`：

- `character_profiles`：初始角色配置
- `player_actions`：每日行动配置
- `story_events`：随机事件和事件选择配置

核心逻辑集中在 `src/game_logic.ts`：

- `create_initial_state`：创建初始游戏状态
- `choose_action`：处理行动选择
- `resolve_event_choice`：处理事件选择
- `acknowledge_exam`：处理考试结果
- `determine_ending`：判定最终结局

界面入口集中在 `src/App.tsx`，像素风样式集中在 `src/styles.css`。

## 扩展方向

- 增加学科细分：语文、数学、英语、物理/历史、化学/政治、生物/地理
- 增加更多长期变量：目标清晰度、老师信任度、自我认同
- 增加更多考试反馈和志愿选择
- 增加事件链，让家庭、人际和手机依赖形成更明显的长期后果
- 增加像素角色动画和更多场景
- 增加结局后的详细成长报告

## 设计原则

- 不把玩家选择写成标准答案
- 不把成绩作为唯一胜利条件
- 不用羞辱式文案描述失败
- 让玩家通过结果感受到节奏、状态和关系的重要性
- 文案保持克制、细腻，贴近考生真实体验
